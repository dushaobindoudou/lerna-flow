## 添加依赖

添加依赖包到指定的包
```
lerna add @lernaflow/init --scope @lernaflow/work
```

添加精确的包版本，如果是项目自依赖最好使用精确版本，因为每次发布新版本
leran 都会自动更新依赖
```
lerna add @lernaflow/init --scope @lernaflow/work --exact
```
